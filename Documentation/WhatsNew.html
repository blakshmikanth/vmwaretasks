1.2 (TBD)

  • 1628: Added support for recording and replaying snapshots.
  • 1633: Added VMWareVirtualMachine.Reset, Suspend, Pause, UnPause, IsPaused and IsSuspended.
  • 1913: Added VMWareVirtualHost.IsConnected.
  • Interface change: split Login into LoginInGuest and WaitForToolsInGuest to be consistent with VIX COM API and allow poweron without tools installed. Exposed a login function with power options.
  • Added support for VMWare Server, added ConnectToVMWareServer.
  • Added VMWareVirtualMachine.ShutdownGuest and PowerOff that specify shutdown parameters explicitly.
  • Bug: GetNamedSnapshot behaves differently on Workstation and VI, now throws exception when snapshot not found. Use FindSnapshotByName to get a null result when the snapshot doesn't exist.
  • Bug: KillProcessInGuest unit test needs to wait for process to actually die according to VMWare docs.
  • Bug: removing a shared folder doesn't remove it from the collection when not the same object.
  • Bug: fixed different behavior of ListDirectoryInGuest between Workstation and VI.
